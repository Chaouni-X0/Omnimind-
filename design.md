# تصميم واجهة مستخدم تطبيق OmniMind الهجين

يهدف هذا التصميم إلى تحديث واجهة المستخدم لتطبيق OmniMind ليصبح عصريًا وفعالًا، مستوحيًا من تطبيقات مثل Manus و Skywork و MiniMax، مع التركيز على تجربة المستخدم بيد واحدة ودعم الاتجاه من اليمين إلى اليسار (RTL) للغة العربية.

## التوجه العام للتصميم

*   **البساطة والأناقة:** واجهة نظيفة وغير مزدحمة، مع التركيز على المحتوى والوظائف الأساسية.
*   **الزجاجية (Glassmorphism):** استخدام تأثيرات الزجاج الشفاف والضبابي لإضفاء عمق وحداثة على الواجهة، خاصة في البطاقات والخلفيات.
*   **الظلال الدقيقة:** استخدام ظلال خفيفة لإبراز العناصر ومنحها إحساسًا بالطبقات.
*   **الرسوم المتحركة السلسة:** انتقالات سلسة بين الشاشات والعناصر لتعزيز تجربة المستخدم.
*   **سهولة الاستخدام بيد واحدة:** وضع عناصر التحكم الرئيسية في متناول الإبهام، خاصة في الجزء السفلي من الشاشة.
*   **دعم RTL:** تصميم جميع الواجهات لدعم اللغة العربية بشكل كامل، مع عكس اتجاه العناصر عند الضرورة.

## قائمة الشاشات الرئيسية

1.  **شاشة لوحة التحكم (Dashboard Screen):**
    *   **المحتوى والوظائف:** عرض المشاريع الحالية، ملخص للنشاط الأخير، إحصائيات سريعة، زر لإنشاء مشروع جديد. بطاقات للمشاريع مع أيقونات وحالة.
    *   **التدفقات الرئيسية:** النقر على مشروع يفتح شاشة تفاصيل المشروع. النقر على زر إنشاء مشروع يفتح شاشة إنشاء مشروع.

2.  **شاشة تفاصيل المشروع (Project Detail Screen):**
    *   **المحتوى والوظائف:** عرض تفاصيل المشروع، قائمة بالمهام (Tasks)، الوصول إلى الترمنال، محرر النصوص، وغرف الدردشة الخاصة بالمشروع. أزرار للتحكم في المشروع (تشغيل، إيقاف، إعدادات).
    *   **التدفقات الرئيسية:** التنقل بين علامات التبويب (Tasks, Terminal, Editor, Chat). النقر على مهمة يفتح تفاصيل المهمة.

3.  **شاشة الترمنال (Terminal Screen):**
    *   **المحتوى والوظائف:** واجهة سطر أوامر تفاعلية، عرض مخرجات الأوامر، إدخال الأوامر. دعم التلوين والتمييز.
    *   **التدفقات الرئيسية:** إدخال أمر وتنفيذه، عرض النتائج.

4.  **شاشة محرر النصوص (Text Editor Screen):**
    *   **المحتوى والوظائف:** عرض وتحرير الملفات النصية (كود، markdown، إلخ). دعم تمييز بناء الجملة (Syntax Highlighting) وأرقام الأسطر. قائمة بالملفات في المشروع.
    *   **التدفقات الرئيسية:** فتح ملف للتحرير، حفظ التغييرات، إنشاء ملف جديد.

5.  **شاشة الدردشة (Chat Screen):**
    *   **المحتوى والوظائف:** عرض رسائل الدردشة (بين المستخدم والوكلاء، أو بين الوكلاء). إدخال رسائل جديدة. دعم عرض أنواع مختلفة من الرسائل (نص، كود، نتائج).
    *   **التدفقات الرئيسية:** إرسال رسالة، التفاعل مع رسائل الوكلاء.

6.  **شاشة إعدادات GitHub (GitHub Settings Screen):**
    *   **المحتوى والوظائف:** ربط حساب GitHub، عرض المستودعات، إعدادات التكامل (مثل Webhooks). أزرار للمزامنة، سحب (Pull)، ودفع (Push) التغييرات.
    *   **التدفقات الرئيسية:** تسجيل الدخول إلى GitHub، اختيار مستودع للربط.

7.  **شاشة الإعدادات العامة (General Settings Screen):**
    *   **المحتوى والوظائف:** إعدادات الحساب، المظهر (السمات)، اللغة، الإشعارات.
    *   **التدفقات الرئيسية:** تغيير السمة، تبديل اللغة.

## تدفقات المستخدم الرئيسية (Key User Flows)

*   **إنشاء مشروع جديد:** Dashboard -> Create New Project -> Project Detail Screen.
*   **العمل على مشروع:** Dashboard -> Select Project -> Project Detail Screen -> (Terminal / Editor / Chat).
*   **إدارة GitHub:** Project Detail Screen -> GitHub Settings -> Sync/Pull/Push.

## خيارات الألوان (Color Choices)

سنعتمد على لوحة ألوان حديثة ومريحة للعين، مع دعم الوضع الفاتح والداكن، مستوحاة من تطبيقات Manus و Skywork و MiniMax.

*   **الألوان الأساسية (Primary Colors):**
    *   **اللون الأساسي (Primary):** أزرق داكن/بنفسجي (مثل `#6200EE` أو `#7B61FF`) لزر الإجراءات الرئيسية والعناصر النشطة.
    *   **اللون الثانوي (Secondary):** لون مكمل فاتح (مثل `#03DAC6` أو `#00C853`) للعناصر الثانوية أو التمييز.

*   **ألوان الخلفية والسطح (Background & Surface Colors):**
    *   **الوضع الفاتح:** خلفية بيضاء ناعمة (`#FFFFFF`)، أسطح بطاقات رمادية فاتحة جدًا (`#F5F5F5`).
    *   **الوضع الداكن:** خلفية رمادية داكنة جدًا (`#121212` أو `#1A1A1A`)، أسطح بطاقات رمادية أغمق قليلاً (`#2C2C2C` أو `#333333`).

*   **ألوان النصوص (Text Colors):**
    *   **الوضع الفاتح:** نص أساسي أسود/رمادي داكن (`#212121`)، نص ثانوي رمادي متوسط (`#757575`).
    *   **الوضع الداكن:** نص أساسي أبيض/رمادي فاتح (`#E0E0E0`)، نص ثانوي رمادي فاتح (`#B0B0B0`).

*   **ألوان الحالة (Status Colors):**
    *   **نجاح (Success):** أخضر (`#4CAF50`)
    *   **تحذير (Warning):** برتقالي (`#FFC107`)
    *   **خطأ (Error):** أحمر (`#F44336`)

*   **ألوان الترمنال ومحرر النصوص:**
    *   **خلفية الترمنال:** أسود داكن (`#1E1E1E`)
    *   **نص الترمنال:** أخضر فاتح (`#00FF00`) أو أبيض (`#F8F8F8`)
    *   **تمييز بناء الجملة (Syntax Highlighting):** ألوان متباينة للأكواد (أزرق، أصفر، أحمر، إلخ).

سيتم تطبيق هذه الألوان باستخدام نظام الثيمات في Jetpack Compose لدعم التبديل بين الوضع الفاتح والداكن بسلاسة.
