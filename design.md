# تصميم واجهة المستخدم - OmniMind

## وثيقة تصميم للواجهة الهجينة (Hybrid) وتطبيق Android

تصف هذه الوثيقة التوجه العام لتصميم واجهة OmniMind.

## التوجه العام

- **Glassmorphism**: تأثيرات الزجاج الشفاف
- **سهولة الاستخدام بيد واحدة**: عناصر التحكم في متناول الإبهام
- **دعم RTL**: للغة العربية
- **رسوم متحركة سلسة**

## الشاشات

1. **Dashboard**: عرض المشاريع والنشاط الأخير
2. **Project Detail**: تفاصيل المشروع والمهام
3. **Terminal**: واجهة أوامر تفاعلية
4. **Editor**: محرر نصوص مع تمييز بناء الجملة
5. **Chat**: دردشة مع الوكلاء
6. **GitHub Settings**: ربط GitHub
7. **General Settings**: المظهر واللغة

## لوحة الألوان

- **الأساسي**: أزرق داكن/بنفسجي
- **الثانوي**: لون مكمل فاتح
- **الخلفية**: أبيض ناعم (فاتح) / رمادي داكن (داكن)
- **الحالة**: أخضر (نجاح)، برتقالي (تحذير)، أحمر (خطأ)

## ملاحظات التنفيذ

- تطبيق Android يستخدم Jetpack Compose للثيمات (Obsidian, Aurora, Ember)
- الواجهة الهجينة (Expo) تستخدم NativeWind مع ثيمات مختلفة
- الشاشات موزعة بين المشروعين حسب المنصة المستهدفة
