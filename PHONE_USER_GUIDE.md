# دليل مستخدم OmniMind من الهاتف

## الهدف

هذا الدليل يشرح كيفية تحميل وتثبيت واستخدام تطبيق OmniMind Android من الهاتف.

---

## البدء السريع

### الخطوة 1: تحميل APK

**من GitHub Actions:**
1. افتح المستودع على GitHub
2. اذهب إلى **Actions** > اختر آخر build
3. حمّل artifact **OmniMind-debug-apks**
4. فك الضغط واختر APK المناسب لجهازك (universal يعمل على معظم الأجهزة)

### الخطوة 2: تثبيت التطبيق

1. افتح مدير الملفات
2. ابحث عن APK الذي نزّلته
3. انقر عليه واختر **Install**

### الخطوة 3: تشغيل التطبيق

1. ابحث عن **OmniMind** في تطبيقاتك
2. اتبع شاشة الإعداد الأولى (اختيار السمة، إضافة مفتاح API، تحديد الميزانية)

---

## الميزات الرئيسية

### 1. إضافة مفاتيح API
- اذهب إلى شاشة **API Center**
- أضف مفتاحاً من: Gemini, OpenAI, Anthropic, OpenRouter, Groq, أو مزود مخصص

### 2. إنشاء مشروع
- من **Dashboard**، أنشئ مشروعاً جديداً باسم
- سيتم إنشاء workspace في مساحة التطبيق

### 3. استخدام Swarm
- من **Chat**، أدخل مهمة لـ Swarm
- النماذج الخمسة (Architect, Analyst, Coder, Tester, Guardian) ستعمل بالتسلسل
- شاهد التقدم والردود

### 4. محرر النصوص
- استعرض ملفات ومجلدات workspace
- أنشئ، افتح، عدّل، احفظ الملفات

### 5. Terminal
- أوامر مدمجة: help, pwd, ls, cd, cat, echo, mkdir, touch, rm, clear, history
- أوامر Git للقراءة: status, log, diff, branch, show

### 6. GitHub
- أدخل Personal Access Token
- استعرض مستودعاتك وملفاتها
- استورد ملفات أو حمّل مستودعاً كاملاً
