# 📱 دليل مستخدم OmniMind من الهاتف

## 🎯 الهدف

هذا الدليل يشرح كيفية استخدام تطبيق OmniMind 1x من الهاتف فقط، بدون الحاجة إلى كمبيوتر.

---

## 🚀 البدء السريع (5 دقائق)

### الخطوة 1: تحميل APK

**من GitHub:**
1. افتح: `github.com/Chaouni-X0/Omni-mind-1x-`
2. انقر على **Releases** (أو انتظر آخر بناء في Actions)
3. حمّل `app-debug.apk`

**أو من Releases:**
1. انقر على **Releases** على اليمين
2. حمّل أحدث إصدار

### الخطوة 2: تثبيت التطبيق

1. افتح مدير الملفات
2. انتقل إلى **Downloads**
3. ابحث عن `app-debug.apk`
4. انقر عليه
5. اختر **Install**

**ملاحظة**: قد تحتاج إلى تفعيل "تثبيت من مصادر غير معروفة"
- اذهب إلى Settings > Security
- فعّل "Unknown Sources"

### الخطوة 3: تشغيل التطبيق

1. ابحث عن **OmniMind** في تطبيقاتك
2. انقر عليه لتشغيله
3. استمتع! 🎉

---

## 📚 الميزات الرئيسية

### 1. إضافة API والنماذج

**إضافة مفتاح API:**
1. انقر على **Settings** ⚙️
2. اختر **API Keys**
3. انقر **Add New**
4. أدخل:
   - الاسم: `My Gemini API`
   - المزود: `Google Gemini`
   - المفتاح: `your_api_key_here`
5. انقر **Save**

**إضافة نموذج:**
1. انقر على **Models**
2. انقر **Add Model**
3. أدخل:
   - الاسم: `Architect`
   - النوع: `Architect`
   - المزود: `Gemini`
   - اختر API Key
4. انقر **Create**

### 2. بدء مهمة جديدة

1. انقر على **New Task** ➕
2. أدخل:
   - العنوان: `بناء نظام دفع`
   - الوصف: `نريد نظام دفع آمن`
   - اختر النماذج
3. انقر **Start**

### 3. نقاش بين النماذج

1. بعد بدء المهمة، ستبدأ النماذج بالنقاش
2. اقرأ الآراء المختلفة
3. انقر **Approve** أو **Reject**
4. النماذج ستبدأ العمل

### 4. اختبار الأمان

1. انقر على **Security** 🔒
2. انقر **Run Tests**
3. انتظر النتائج
4. اقرأ الثغرات المكتشفة
5. انقر **Apply Fixes** لإصلاح تلقائي

### 5. تصدير النتائج

1. انقر على **Export** 📤
2. اختر الصيغة:
   - PDF (للتقارير)
   - JSON (للبيانات)
   - CSV (للجداول)
   - ZIP (لملفات متعددة)
3. انقر **Export**
4. سيتم حفظ الملف

## 🔧 الإعدادات

### قاعدة البيانات

1. اذهب إلى **Settings** ⚙️
2. اختر **Database**
3. اختر نوع قاعدة البيانات:
   - SQLite (محلي)
   - MySQL (سحابي)
   - MongoDB (NoSQL)
4. أدخل بيانات الاتصال
5. انقر **Connect**

### الواجهة

1. اذهب إلى **Settings** ⚙️
2. اختر **Appearance**
3. اختر اللون:
   - Obsidian (أسود)
   - Aurora (أزرق)
   - Ember (أحمر)
4. اختر الحجم:
   - صغير
   - وسط
   - كبير

### اللغة

1. اذهب إلى **Settings** ⚙️
2. اختر **Language**
3. اختر:
   - العربية 🇸🇦
   - English 🇬🇧
   - Français 🇫🇷

## 💡 نصائح وحيل

### نصيحة 1: استخدم Terminal

1. انقر على **Terminal** 💻
2. اكتب الأوامر مباشرة
3. اضغط Enter
4. اقرأ النتائج

### نصيحة 2: استخدم محرر النصوص

1. انقر على **Editor** ✏️
2. اكتب الكود
3. اختر اللغة (Python, JavaScript, etc)
4. انقر **Run** لتنفيذه

### نصيحة 3: تصدير متعدد

1. اختر عدة صيغ في نفس الوقت
2. انقر **Export All**
3. سيتم إنشاء ZIP بجميع الملفات

### نصيحة 4: مراقبة الأداء

1. اذهب إلى **Dashboard**
2. اقرأ الإحصائيات:
   - عدد المهام المكتملة
   - معدل النجاح
   - التكلفة الإجمالية
   - الـ tokens المستخدمة

## 🐛 حل المشاكل الشائعة

### المشكلة: التطبيق لا يفتح

**الحل:**
1. تأكد من تثبيت APK بنجاح
2. جرّب إعادة التشغيل
3. احذف التطبيق وثبّته مرة أخرى

### المشكلة: API Key لا يعمل

**الحل:**
1. تأكد من صحة المفتاح
2. تأكد من اتصال الإنترنت
3. جرّب API Key آخر

### المشكلة: النماذج بطيئة

**الحل:**
1. قلل عدد النماذج
2. استخدم نماذج أخف
3. تحقق من الاتصال

### المشكلة: لا يمكن تصدير الملفات

**الحل:**
1. تأكد من وجود مساحة حرة
2. تحقق من أذونات الملفات
3. جرّب صيغة أخرى
