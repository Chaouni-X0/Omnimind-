# OmniMind Hybrid Application v2.0

**OmniMind Hybrid** هو تطبيق متعدد المنصات يجمع بين تطبيق جوال حديث (Expo/React Native) وخادم Node.js قوي، مما يوفر تجربة مستخدم احترافية مع إمكانيات متقدمة للترمنال ومحرر النصوص والتكامل الكامل مع GitHub.

## البنية الهجينة

المشروع مقسم إلى جزأين رئيسيين:

### الواجهة الأمامية (Frontend) - `frontend/`

تطبيق جوال متعدد المنصات مبني باستخدام **Expo** و **React Native**، يعمل على:

- **iOS** - تطبيق أصلي عبر Expo
- **Android** - تطبيق أصلي عبر Expo (ينتج ملف APK)
- **Web** - تطبيق ويب عبر Expo Web

**المميزات:**

- واجهة مستخدم عصرية مستوحاة من Manus و Skywork و MiniMax
- دعم كامل للغة العربية (RTL)
- نظام ثيمات متقدم (فاتح/داكن)
- رسوم متحركة سلسة
- سهولة الاستخدام بيد واحدة

### الواجهة الخلفية (Backend) - `backend/`

خادم Node.js قوي يوفر:

- **API RESTful** للتطبيق الأمامي
- **الترمنال الحقيقي** - تنفيذ الأوامر وإدارة العمليات
- **محرر النصوص** - قراءة وكتابة الملفات مع تمييز بناء الجملة
- **تكامل GitHub** - المزامنة والسحب والدفع
- **إدارة المشاريع** - إنشاء وحذف وتعديل المشاريع
- **نظام الدردشة** - إدارة الرسائل والمحادثات

## البدء السريع

### المتطلبات

- **Node.js** 18+ و **npm** أو **pnpm**
- **Git**
- **Expo CLI** (للتطوير على الجوال)

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/Chaouni-X0/Omni-mind-1x-.git
cd Omni-mind-1x-

# تثبيت المتعلقات
npm install
# أو
pnpm install
```

### التطوير

```bash
# تشغيل الواجهة الأمامية والخلفية معًا
npm run dev

# أو تشغيل كل واحدة على حدة
npm run dev:frontend  # الواجهة الأمامية على http://localhost:8081
npm run dev:backend   # الخادم على http://localhost:3000
```

### البناء

```bash
# بناء الإصدار الإنتاجي
npm run build

# بناء الواجهة الأمامية فقط
npm run build:frontend

# بناء الخادم فقط
npm run build:backend
```

### الاختبار

```bash
# تشغيل جميع الاختبارات
npm run test

# تشغيل اختبارات الواجهة الأمامية فقط
npm run test:frontend

# تشغيل اختبارات الخادم فقط
npm run test:backend
```

## الميزات الرئيسية

### 1. الترمنال المتقدم

- تنفيذ الأوامر في الوقت الفعلي
- عرض مخرجات الأوامر مع التلوين
- دعم الأوامر المعقدة والعمليات الطويلة
- إدارة العمليات (إيقاف، إعادة تشغيل)

### 2. محرر النصوص

- تحرير الملفات مع الحفظ التلقائي
- تمييز بناء الجملة لعدة لغات برمجية
- أرقام الأسطر والبحث والاستبدال
- دعم الملفات الكبيرة

### 3. تكامل GitHub

- المصادقة عبر OAuth
- عرض المستودعات والفروع
- السحب والدفع والدمج
- إدارة الـ Pull Requests
- عرض السجل (History)

### 4. نظام الدردشة

- محادثات في الوقت الفعلي
- دعم الرسائل الغنية (نص، كود، صور)
- إشعارات فورية
- حفظ السجل

### 5. إدارة المشاريع

- إنشاء مشاريع جديدة
- تنظيم الملفات والمجلدات
- تتبع التغييرات
- إحصائيات المشروع
