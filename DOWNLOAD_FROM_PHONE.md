# 📱 تحميل واستخدام التطبيق من الهاتف فقط

## 🎯 الطريقة الأسهل: GitHub Actions

لا تحتاج إلى كمبيوتر! البناء يتم تلقائياً على خوادم GitHub.

---

## 📥 خطوات التحميل من الهاتف

### الخطوة 1: فتح GitHub من الهاتف

```
https://github.com/Chaouni-X0/Omni-mind-1x-
```

### الخطوة 2: الذهاب إلى Actions

1. افتح المستودع
2. انقر على **Actions** (الأيقونة الثالثة)
3. ستجد آخر بناء في القائمة

### الخطوة 3: تحميل APK

1. انقر على آخر بناء (Build)
2. انزل للأسفل إلى **Artifacts**
3. حمّل **app-debug** أو **app-release**

### الخطوة 4: تثبيت التطبيق

1. افتح مدير الملفات
2. انتقل إلى مجلد التحميلات
3. ابحث عن `app-debug.apk`
4. انقر عليه لتثبيته

## 🔧 الطريقة البديلة: استخدام Termux

إذا كنت تريد البناء مباشرة على الهاتف:

### التثبيت

```bash
# 1. ثبّت Termux من F-Droid أو Google Play
# 2. افتح Termux وشغّل:

pkg update
pkg upgrade
pkg install git openjdk-17 gradle
```

### البناء

```bash
# استنساخ المشروع
git clone https://github.com/Chaouni-X0/Omni-mind-1x-.git
cd Omni-mind-1x-

# البناء
chmod +x gradlew
./gradlew assembleDebug
```

### النتيجة

```
app/build/outputs/apk/debug/app-debug.apk
```

## 🐛 استكشاف الأخطاء

### المشكلة: لا أجد Artifacts

**الحل:**
1. تأكد من أن البناء نجح (✅ أخضر)
2. انزل للأسفل في صفحة البناء
3. ستجد Artifacts هناك

### المشكلة: APK لم يثبّت

**الحل:**
1. تأكد من تفعيل "تثبيت من مصادر غير معروفة"
2. جرّب app-debug.apk بدلاً من app-release
3. تأكد من توافق الإصدار (Android 8.0+)

### المشكلة: Termux بطيء جداً

**الحل:**
1. استخدم GitHub Actions بدلاً منه
2. أو استخدم Codemagic
3. أو استخدم كمبيوتر إذا أمكن
